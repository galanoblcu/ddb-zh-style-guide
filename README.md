# DDB中文技术文档写作风格指南

该指南以<https://zh-style-guide.readthedocs.io>为基础，根据DDB用户手册及教程的实际需要相应修订。

感谢原作者对原仓库《中文技术文档写作风格指南》的源文件的分享！

## 如何贡献

所有的源文件都存放在 /source 文件夹下：

- 配置文件为 conf.py 文件
- 各章节目录为各个 index.rst 文件
- 各章节页面为各个 xxx.md 文件

如需贡献，**请直接修改 /source 文件夹下的相应文件。**

## 用途

仅限DDB用户手册及教程。
